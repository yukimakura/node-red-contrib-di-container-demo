# NodeRedの自作ノードでTsyringeをつかうデモ
```
yarn install
yarn run build
```
でパッケージを作成できます。

あとは、NodeRedの`flows.js`に移動して、`yarn add このぷろじぇくとのでぃれくとり`をし、NodeRedを再起動してください
